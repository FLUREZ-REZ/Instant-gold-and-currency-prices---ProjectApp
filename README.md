# Instant-gold-and-currency-prices---ProjectApp
An application with login and authentication with an SMS verification code and the ability to instantly view the price of dollars and gold and various currencies, and the application is bilingual in Persian and English, and other features such as dark and light mode, etc.

What was used in this project ? : 

- shared_preferences  
- flutter_secure_storage  
- provider  
- http
- shimmer
- clean architecture
- APIS
- directionality
- price API
- sms otp verification
- etc packages
    

  یک برنامه با ورود و احراز هویت با کد تایید پیامکی و امکان مشاهده لحظه ای قیمت دلار و طلا و انواع ارز ها و دارای دو زبانه بودن برنامه به صورت فارسی و انگلیسی و دیگر قابلیت هایی همچون مود تاریک و روشن و ...



          
  This project is intended to create a resume and practice and is of no value.    
  این پروژه به منظور ایجاد یک رزومه وتمرین بوده برای یادگیری و پیشرفت برای پروژه و تمرینات دیگر
